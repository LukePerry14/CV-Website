Projects

Some notable independent projects I've worked on - code avaiable on my github public repositories.

Movein

A social media style app to optimise and gamify the house-mate finding experience to allow individuals and groups to express individuality to pair individuals and groups. I lead development for the full stack - working on frontend, backend, and business aspects of the project.

AI Search - TSP

I use a modified Particle Swarm Optimisation and Simulated Annealling algorithm to efficiently generate heuristically optimal solutions to the Travelling Salesman Problem. This required developing and optimising a pseudo-spatial encoding of tours to allow for tour 'proximities'(similarities) and velocities.

Shortest Vector Problem (C++)

I implemented a vector sieving solution that, while not asymptotically efficient, demonstrates practical efficiency compared to alternative methods for discovering minimal norm lattice points in non-standard high-dimensional vector spaces. This approach is particularly valuable in cryptography and computational geometry.

Regression Model for Wine Quality

I attempt to train a regression model on a wine component dataset with high multicolinearity to generate a hyper-solution-space to predict unseen high quality wine compositions.

GPT Style LLM Model

Developed a bigram language model using PyTorch, implementing tokenization, bigram probability calculation, and text generation to demonstrate foundational NLP concepts. Additionally, designed a GPT-style language model utilizing a Transformer architecture, incorporating self-attention mechanisms and positional encoding, exploring state of the art text generation.

C++ Chess Engine

Developed a high-performance C++ chess engine with efficient board representation, move generation, and evaluation functions. Implemented core architecture, minimax, and alpha-beta pruning algorithms. Implementation focused on optimal storage/ representation (bitboards) and computation - also explored optimisation through a combination of lock-free, lazy symettric multithreading, combined with a principle variation splitting method. Achieved significant ELO rating in online tests and integrated the engine with a CLI. Technologies used: C++, STL, Multithreading, Bitboards, Minimax Algorithm, Alpha-Beta Pruning

DPLL SAT Solver

Developed a robust and efficient SAT (Boolean Satisfiability Problem) solver based on the Davis-Putnam-Logemann-Loveland (DPLL) algorithm to handle complex logical formulae and provide solutions for satisfiability in a range of applications.

File Transfer software

A client-server routed messaging and file transfer application that provides real-time communication and efficient file exchange capabilities between users.
